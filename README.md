# 英雄联盟技能知识库

这是一个可直接发布到 GitHub Pages 的静态页面包。

## 使用方式

1. 将当前目录内容推送到一个公开 GitHub 仓库根目录。
2. 在仓库的 `Settings -> Pages` 中启用 GitHub Pages。
3. 选择 `Deploy from a branch`。
4. 选择 `main` 分支和 `/(root)` 目录。
5. 等待部署完成后访问 Pages 地址。

## 说明

- `index.html` 是默认入口页。
- `champions_github.html` 是同一份页面的备用文件名。
- 视频使用 Riot 官方在线地址，不占用仓库体积。
